Welcome to vinsol memory game

The aim of the game is to match the hide picture pairs. when clicking on a box,it shows the hidden picture. When the next card is matching, you have found a pair. That pair stays uncovered. If the two cards are not matching, it is being covered up again. The game is finished if you uncovered all pairs. Try to do it with as little tries as possible.
